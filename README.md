# Portfolio Website

A professional portfolio website for developers and designers.

## Features

- Responsive design
- Dark/Light mode toggle
- Single page application with smooth scrolling
- Modern CSS with SCSS preprocessing
- Mobile-first approach

## Project Structure

```
portfolio/
├── index.html          # Main HTML file
├── css/               # Compiled CSS files
├── scss/              # SCSS source files
├── js/                # JavaScript files
├── img/               # Image assets
└── vendor/            # Third-party libraries
```

## Development Setup

1. Install dependencies:
```bash
npm install
```

2. Build SCSS files:
```bash
npm run build
```

3. Run development server:
```bash
npm run dev
```

## Technologies Used

- HTML5
- CSS3/SCSS
- JavaScript (ES6+)
- Bootstrap 5
- Devicons
- Font Awesome
- Simple Line Icons

## File Structure

The website uses a component-based approach with:
- Navigation sidebar
- Responsive sections
- Mobile-first design principles
- Accessible color schemes
