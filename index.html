Projetos

Pipeline de KPIs Logísticos — Olist Dataset

Python · PostgreSQL · Docker · Streamlit · Plotly

Primeiro projeto de engenharia de dados construído do zero. Peguei o dataset público de e-commerce brasileiro da Olist e montei um pipeline ELT completo: os dados brutos entram via ingestão com SQLAlchemy, passam por transformações em SQL agrupadas por estado e período, e chegam num dashboard de 6 páginas com mapas coropléticos e filtros dinâmicos. Tudo containerizado com Docker.

O projeto existe porque eu precisava provar pra mim mesmo que consigo fazer o fluxo inteiro — da ingestão até a visualização.

Crypto Monitor — Pipeline de Dados de Criptomoedas

Python · PostgreSQL · dbt · Apache Airflow · Streamlit · Docker

Pipeline de dados cripto ponta a ponta. Coleta candles diários de BTC, ETH, BNB, SOL e ADA via Binance API, câmbio USD/BRL via BACEN e índice Fear & Greed — tudo orquestrado pelo Apache Airflow com execução automática diária. O dbt calcula RSI(14), Médias Móveis, Bollinger Bands, MACD e sinais de alta confluência.

O resultado é um dashboard Streamlit de 3 páginas: análise técnica por ativo, comparação e heatmap de correlação entre ativos, e backtesting de estratégias com curva de capital. Tudo containerizado com Docker.
